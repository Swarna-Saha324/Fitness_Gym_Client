# 🏋️ ApexFit - Fitness Class Booking & Community Platform

ApexFit is a full-stack fitness management platform where users can explore fitness classes, book sessions through Stripe, join community discussions, and manage their fitness journey. Trainers can create and manage classes, while administrators oversee users, trainers, classes, and forum content through a role-based dashboard.

# ✨ Key Features

- 🔐 Better Auth Authentication (Email/Password & Google Login)
- 🍪 JWT Authentication with HTTPOnly Cookies
- 👤 Role-Based Dashboard (User, Trainer & Admin)
- 🏋️ Browse Approved Fitness Classes
- 🔍 Search Classes using MongoDB `$regex`
- 🎯 Filter Classes by Category using MongoDB `$in`
- 💳 Secure Stripe Payment Integration
- 📅 Prevent Duplicate Class Booking
- ❤️ Add & Remove Favorite Classes
- 💪 Trainer Application System with Admin Approval
- 📢 Community Forum with Like/Dislike System
- 💬 Comment, Reply, Edit & Delete Own Comments
- 👨‍🏫 Trainer Class Management
- 👥 View Students Enrolled in Classes
- 📝 Trainer & Admin Forum Post Management
- 🛡️ Admin User Management (Block/Unblock & Make Admin)
- 🏷️ Manage Trainers (Approve, Reject & Demote)
- 📚 Manage Classes (Approve, Reject & Delete)
- 💰 View Stripe Transaction History
- 📄 Server-side Pagination
- 📱 Fully Responsive Design
- 🎨 Modern UI with Framer Motion Animations
- ⏳ Global Loading Spinner
- 🚫 Custom 404 Error Page

# ⚙️ Installation

## Clone Client Repository

```bash
git clone https://github.com/Swarna-Saha324/Fitness_Gym_Client.git
```

## Clone Server Repository

```bash
git clone https://github.com/Swarna-Saha324/fitness_server.git
```

## Install Dependencies

### Client

```bash
cd Fitness_Gym_Client
npm install
npm run dev
```

### Server

```bash
cd fitness_server
npm install
npm run dev
```

# 🔑 Environment Variables

Create a `.env` file and add the following variables:

```env
PORT=
MONGODB_URI=
JWT_SECRET=
BETTER_AUTH_SECRET=
BETTER_AUTH_URL=
STRIPE_SECRET_KEY=
CLIENT_URL=
---

`
